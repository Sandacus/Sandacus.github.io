I have decided to strip back this website and start from scratch! And today is the day I a have finally taken the first steps.

I have removed all the styling and javascript and will build it from the ground up. I have been thinking of doing this for a long time, however procrastination and wanting to have a perfect vision of what I'm aiming for have stopped me. I have realised that this is the wrong way to think! It is far better to start something imperfectly and iterate on that than never to start at all.

I intend to have a section act as a mini blog to help keep myself accountable in creating this website.

  • EJS, a javascript based templating engine.
  • Jekyll, a ruby based templating engine that is advertised for the github pages.
  • Jinja, which a python based templating engine, used in the flask web framework.
